# Spidi 

## Description
Spidi est une application mobile développée en React Native dans le but de passer un test.


1. **Cloner le repository :**
2. **Installer les dépendances**
3. **Lancer l'application :**
- **Sur iOS :**
  ```
  npx react-native run-ios
  ```

- **Sur Android :**
  ```
  npx react-native run-android
  ```
  NB: vous pouvez egalement utilisez : npm start (recommander)
## Technologies utilisées

- React Native
- React Navigation
- Redux (facultatif)
